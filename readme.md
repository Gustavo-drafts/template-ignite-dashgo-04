## 💻 Pré-requisitos

Antes de começar, verifique os seguintes requisitos:

- Requer versões:  

```
rodando com versões atualizadas
```

## 🚀 Instalando **daashgo**

Para instalar o **dashgo**, siga estas etapas:

Win, Linux e macOS:

```
npm i
or
yarn
```

to Run app:

```
npm run dev
or
yarn dev
```
